# React 쇼핑몰


### useState와 useEffect 훅을 적극 사용하여, 동적인 사용자 인터페이스를 구현하였고 
### Redux와 sessionStorage를 이용해 최근본상품,장바구니를 저장하고 관리하여 
### 사용자에게 더욱 맞춤화된 쇼핑 경험을 제공합니다. 
### 반응형웹으로 만들어져 모든 환경에서 사용가능합니다.

### [사이트 바로가기](https://shoeshop-ten.vercel.app/)
